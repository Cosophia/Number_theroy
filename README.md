# Number_theroy
## Introduction:
Actually,i do not know how to or where to start learning this interesting theory in Math.
Luckily,the book named "Millennium Problems" talking about  the Riemann's  Hypothesis,which mentions the Euclid's idea of the prime number.
For Euclid,the number either is a prime or the product of any primes.
So i decide to write a program to find out his idea,and it's the start point of this learning.
### Version 1.0
* We go try to find out the prime number with the O(n^2) time complexity firstly.
* Then we try to cut down the time complexity to the O(nloglogn) with the Sieve of Eratosthenes.
